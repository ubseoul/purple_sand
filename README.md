# Purple Sand

Purple Sand is a personal Reddit-style dashboard for a calmer, higher-signal internet feed.

It is designed around the things Ade actually wants to see:

- Reputable world news
- Al Jazeera headlines
- Anime news
- League of Legends / esports news
- Location-based weather
- A clean card-based interface instead of doomscroll chaos

## MVP

This first version is a static front-end prototype. It can be opened directly in a browser or deployed with GitHub Pages.

## Planned features

- Live RSS/API integrations
- Location-based weather using Open-Meteo or WeatherAPI
- Custom source ranking
- Saved posts
- Personal tags
- “Less chaos” reading mode
- Mobile-first layout

## Files

- `index.html` — app shell and feed cards
- `styles.css` — visual design
- `app.js` — tab filtering, source data, weather placeholder

## Deployment

Enable GitHub Pages from the repository settings and serve from the `main` branch.
